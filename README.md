# Koffin - Cafetería de Especialidad

Este es el repositorio del sitio web de Koffin, desarrollado como un proyecto web responsivo.

## 🛠️ Tecnologías Aplicadas e Implementación
- **HTML:** Uso correcto de componentes estructurales (`<header>`, `<nav>`, `<main>`, `<section>`, `<footer2>`).
- **Formulario Formspree:** Procesador de contactos activo y funcional.
- **Flexbox Layout:** Utilizado dinámicamente en la sección **"Menú"** para que las tarjetas de categorías fluyan de manera elástica según las dimensiones del dispositivo.
- **CSS Grid:** Implementado en la sección **"Reseñas"** para construir una grilla sólida de opiniones reales de Google Maps.
- **Media Queries:** Definidos estratégicamente para colapsar menús de navegación, contenedores de contacto y las vistas de los mapas bidireccionales en smartphones y tablets.
